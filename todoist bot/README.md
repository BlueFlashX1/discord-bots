# Todoist Bot

Discord bot integration with Todoist for task management.

## Features

- Create and manage Todoist tasks from Discord
- Task synchronization
- Reminders and notifications
- Project and label management

## Setup

1. Copy `.env.example` to `.env` and fill in your Discord bot token and Todoist API key
2. Run `npm install` to install dependencies
3. Run `npm run deploy` to register slash commands
4. Run `npm start` to start the bot

## Environment Variables

See `.env.example` for required variables.
