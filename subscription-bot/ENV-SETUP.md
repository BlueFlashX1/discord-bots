# Environment Variables Setup

Create a `.env` file in this directory with the following variables:

```bash
# Discord Bot Configuration
DISCORD_TOKEN=your_discord_bot_token_here
CLIENT_ID=your_discord_client_id_here
```

## Required Variables

- **DISCORD_TOKEN** - Your Discord bot token (required)
- **CLIENT_ID** - Your Discord application client ID (required for command deployment)

## Setup Steps

1. Create `.env` file with your actual values
2. Get your bot token from: https://discord.com/developers/applications
3. Get your client ID from the same page (Application ID)
