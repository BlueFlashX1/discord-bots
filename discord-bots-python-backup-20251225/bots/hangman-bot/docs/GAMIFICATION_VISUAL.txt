## Scoring System (Examples)

```
WIN on 4-letter word with 0 mistakes:
  Base (100) + Word bonus (40) + Perfect (50) = 190 pts

WIN on 7-letter word with 2 mistakes:
  Base (100) + Word bonus (70) + Accuracy (30) - Penalty (40) = 160 pts

WIN on 10-letter word with 5 mistakes:
  Base (100) + Word bonus (100) + Accuracy (15) - Penalty (100) = 115 pts

LOSS on any word:
  0 points

â†’ Encourages accuracy, difficulty, and quick solving!
```
