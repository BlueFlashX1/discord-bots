# Coding Practice Bot

Discord bot for Python coding practice with problems from multiple sources (LeetCode, Codewars, etc.).

## Features

- Daily coding challenges
- Problem submission and validation
- Progress tracking
- Solutions archive
- User preferences and settings

## Setup

1. Copy `.env.example` to `.env` and fill in your Discord bot token
2. Run `npm install` to install dependencies
3. Run `npm run deploy` to register slash commands
4. Run `npm start` to start the bot

## Environment Variables

See `.env.example` for required variables.
