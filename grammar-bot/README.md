# Grammar Bot (Mr. Hall)

Discord bot that helps users improve their grammar and writing skills.

## Features

- Grammar checking and corrections
- Writing feedback
- Educational tips and suggestions
- User progress tracking

## Setup

1. Copy `.env.example` to `.env` and fill in your Discord bot token
2. Run `npm install` to install dependencies
3. Run `npm run deploy` to register slash commands
4. Run `npm start` to start the bot

## Environment Variables

See `.env.example` for required variables.
