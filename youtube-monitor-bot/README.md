# YouTube Monitor Bot

Discord bot that monitors YouTube channels and posts new videos.

## Features

- YouTube channel monitoring
- New video notifications
- Channel management (add/remove)
- Customizable notification settings
- API quota management

## Setup

1. Copy `.env.example` to `.env` and fill in your Discord bot token and YouTube API key
2. Run `npm install` to install dependencies
3. Run `npm run deploy` to register slash commands
4. Run `npm start` to start the bot

## Environment Variables

See `.env.example` for required variables.
